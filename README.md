# Strompreis24 Website

Dies ist eine statische Website für Strompreis24, erstellt mit HTML, CSS und JavaScript.

## Schnellstart

1. Projektordner öffnen:
   - Öffne den Ordner `Strompreis24_website` in deinem Editor.
2. Website lokal starten:
   - Öffne `index.html` direkt im Browser.
3. Dateien bearbeiten:
   - HTML-Dateien im Projektstamm bearbeiten.
   - Gemeinsame Stile in `assets/css/style.css` anpassen.
   - Gemeinsame Interaktionen in `assets/js/main.js` anpassen.
4. Website online stellen:
   - Die komplette Struktur ist statisch und kann bei jedem Webhoster oder GitHub Pages bereitgestellt werden.

## Struktur

- `index.html` – Startseite
- `strom-gas-angebote.html` – Angebote
- `tarifrechner.html` – Tarifrechner
- `vertriebspartner.html` – Partnerseite
- `ueber-uns.html` – Über uns
- `kontakt.html` – Kontaktseite
- `impressum.html` – Impressum
- `datenschutz.html` – Datenschutz
- `assets/css/style.css` – zentrale Styles
- `assets/js/main.js` – zentrales JavaScript
- `assets/img/` – Bilder und Icons
